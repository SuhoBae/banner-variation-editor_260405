# 완성 체크리스트

## 제품 완성 기준
- 선택, 이동, 리사이즈, 직접 편집이 레이어 타입별로 일관된다.
- 보드별 독립 수정과 원본 복귀가 명확하다.
- export 결과가 화면과 충분히 동일하다.
- 에셋 관리와 교체가 빠르다.
- 작은 배너와 큰 배너 모두에서 텍스트가 무너지지 않는다.
- 다른 기기에서 clone 후 바로 실행 가능하다.
